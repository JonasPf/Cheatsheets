Markdown

Basics

Paragraphs

Paragraph are separated by blank lines: Paragrah1 Paragraph2

Headers

# H1 ## H2 ### H3 #### H4 ##### H5 ###### H6 or H1 ====== H2 ------

Text Elements

*text* or _text_
Italic: text
**text** or __text__
Bold: text
~~text~~
Strikethrough: text
--- (three or more)
Horizontal line
\*literal asterisks\*
Use backslash to escape Markdown commands

Lists

Unordered lists

* Red * Green * Blue or + Red + Green + Blue or - Red - Green - Blue

Ordered lists

1. Bird 2. McHale 3. Parish or (for the lazy) 1. Bird 1. McHale 1. Parish

Ordered lists

Must be indented by a tab or four spaces: - Colour - Red - Blue - Animal - Dog - Cat

Links

Inline:

  • [Example with title](http://example.com/ "Optional Title")
  • [Example link](http://example.net/)

Reference:

  • Link: This is [an example] [id] referencence-style link.
  • Reference: [id]: http://example.com/ "Optional Title Here"

Implicit reference:

  • Link: [Google][]
  • Reference: [Google]: http://google.com/

Automatic link:

  • <http://example.com/>
  • <address@example.com/>

Code

This is a normal paragraph: This is a code block.

Images

Inline:

  • ![Alt text](/path/to/img.jpg)
  • ![Alt text](/path/to/img.jpg "Optional title")

Reference:

  • Link: ![Alt text][id]
  • Reference: [id]: url/to/image "Optional title attribute"
