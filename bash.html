Configuration

/etc/profile
generic system-wide profile
.bash_profile
processed every time I login (e.g. via ssh, console, ...)
.bashrc
processed every time I open a non-login shell (e.g. xterm)

Call .bashrc from .bash_profile: if [ -f ~/.bashrc ] then ~/.bashrc fi

Keyboard Shortcuts

Ctrl + A Go to the beginning of the line
Ctrl + E Go to the end of the line
Ctrl + L Clear the screen
Ctrl + R Search through previously used commands
Ctrl + Z Puts the current process into the background. Run fg to restore it.
Ctrl + K Delete everything after the cursor
Ctrl + U Delete everything before the cursor
Ctrl + T Swap the two characters before the cursor
Tab Autocomplete

Alias

alias ls="ls -l"
Calls ls -l every time I run ls
unalias ls
Removes previous alias
alias -p
Print all aliases

Processes

jobs
Lists all background processes and there jobspec
ps
Lists all processes and there pid
Ctrl + Z
Puts the current process into the background
fg [%jobspec]
Puts process in the foreground
bg [%jobspec]
Puts process in the background
cp bigfile target &
Runs copy process in the background
Ctrl + C
Kill the current foreground process
kill -15 %jobspec|pid
Shutdown the process orderly.
kill -9 %jobspec|pid
Kill the process mercilessly

Quoting

Backslash: \
Escape character
Single quote: 'string'
Every character stands for itself. No escape character
Double quote: "string"
$, `, \, !, * and @ have a special meaning
ANSI-C$'string'
